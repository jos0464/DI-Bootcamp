Description

The PCAS (Personalized Content Analysis System) project is a Python application that:

Generates a dataset of 50 simulated users with:

15 different activities corresponding to 15 interests (technology, fitness, music, art, gaming, science, reading, cooking, sports, photography, travel, movies, writing, gardening, dancing).

Users’ ages and their performed activities.

Analyzes the data to provide:

Distribution of interests among users.

Most popular activities.

Basic statistics such as mean and median age.

Generates personalized recommendations for each user:

Each user receives 3 new activity recommendations based on the average popularity of activities among other users for interests they haven’t explored yet.

Visualizes the data with:

Horizontal bar chart of interests.

Pie chart of interests.

Activity heatmap per user.
